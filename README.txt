This is a lightweight minecraft mod which helps you sort the enchantments on items' tooltip.

Moreover, some misc fuctions related are provided.

You can configure all the functions at your want.

Mod Files now available on [curseforge](https://www.curseforge.com/minecraft/mc-mods/enchantment-sort/files) and [mcmod](https://www.mcmod.cn/download/8442.html)
